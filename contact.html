Ce script est conçu pour faciliter le processus de calcul des paramètres d'usinage CNC pour une fraise. Voici un résumé de ses principales fonctionnalités :

Interface Utilisateur : Le script crée une interface utilisateur avec des champs de formulaire pour saisir les détails de l'outil de coupe, tels que le diamètre de la fraise, le matériau, le nombre de dents, etc.

Calcul Automatique : Il effectue automatiquement des calculs en fonction des paramètres saisis, notamment la vitesse de coupe (Vc), l'avance par dent (fz), la profondeur de passe (ap), etc.

Réglages Personnalisés : L'utilisateur peut spécifier des paramètres personnalisés tels que la vitesse de coupe, le diamètre de l'outil, etc. De plus, il peut définir des options facultatives telles que la coupe au centre, l'orientation de la coupe, etc.

Génération de Fichier JSON : Le script génère un fichier JSON structuré avec toutes les données de l'outil de coupe. Ce fichier peut être utilisé pour stocker et échanger des configurations d'outils.

Réactivité : L'interface utilisateur est réactive, fournissant des suggestions en temps réel et permettant à l'utilisateur de générer rapidement des données utiles pour l'usinage CNC.

Affichage des Résultats : Les résultats des calculs, y compris la vitesse de broche, la vitesse de coupe, l'avance, etc., sont affichés de manière claire dans un tableau.

Réduction Automatique de la Vitesse de Broche : Si les paramètres spécifiés dépassent les capacités de la machine, le script ajuste automatiquement la vitesse de broche pour rester dans les limites de la machine.

Résumé des Performances : Le script produit un résumé des performances, indiquant les ajustements effectués et fournissant des recommandations, le cas échéant.

En résumé, ce script offre une solution complète pour le calcul des paramètres d'usinage CNC, simplifiant le processus pour les utilisateurs travaillant avec des fraises de différentes tailles et dans divers matériaux.
